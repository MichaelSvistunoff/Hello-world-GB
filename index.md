## Стандартные команды [Main commands]

### Подготовка репозитория [repository's preparing]

Для создание репозитория необходимо выполнить команду: 
<pre>git init</pre>  
в папке с репозиторием и у Вас создаться репозиторий (появится скрытая папка <mark>.git</mark>)

### Создание коммитов [commit creation]

#### Подготовка к коммиту [Git add]

***Для добавления измений*** в коммит используется команда `git add`. Чтобы использовать команду git add напишите <pre>`git` add <file_name.extension></pre>

\* Чтобы не печатать полностью название файла, нужно ввести первые две буквы его названия и нажать **Tab**

#### Создание коммитов [commit creation]

Для того, чтобы создать коммит(сохранение) необходимо выполнить команду `git commit`. Выполняется она так: <pre>`git` commit -m "<commit_message></pre> Все файлы для коммита должны быть **ДОБАВЛЕНЫ** и сообщение к коммиту писать **ОБЯЗАТЕЛЬНО**

### Просмотр состояния репозитория [view repository's status]

Для того, чтобы посмотреть состояние репозитория используется команда `git status`. Для этого необходимо в папке с репозиторием написать <pre>`git` status</pre> и Вы увидите были ли измения в файлах, или их не было

### Перемещение между сохранениями [choosing save]

Для того, чтобы перемещаться между коммитами, используется команда `git checkout`. Используется она в папке с пепозиторием следующим образом: <pre>`git` checkout <commit_number></pre>

\* Можно скопировать 4 первые символа нужного коммита

### Журнал изменений [changes log]

Для того, чтобы посмтреть все сделанные изменения в репозитории, используется команда `git log`. Для этого достаточно выполнить команду <pre>`git` log</pre> в папке с репозиторием

## Ветки в Git [branches in Git]

### Создание ветки [branch's creation]

Для того, чтобы создать ветку, используется команда `git branch`. Делается это следующим образом в папке с репозиторием: 
<pre>`git` branch <new_branch_name></pre>

### Слияние веток [branch's merge]

Для того чтобы дабавить ветку в текущую ветку используется команда <pre>`git merge <name_branch>`</pre>

### Удаление веток [branch's delete]

Для удаления ветки ввести команду <pre>`git` branch -d 'name branch'</pre>
